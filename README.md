# Internsip Task2-Student-Management-System
## Problem Statement:- Create a simple program to manage students using Java OOP concepts.
### Explanation
In Task-2, I have created an console application which performs operation like adding student, removing student and display student basically an curd app, In this task i have learned about OOP's concept, collection framework, looping, conditonal statement and many more things.
### Evaluation
In future we can add database concept for storing this Students detail permenantly using JDBC or Hibernate.
